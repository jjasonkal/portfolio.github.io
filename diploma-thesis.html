Semantic image segmentation using depth estimation

Machine Learning Computer Vision Neural Networks Python PyTorch Linux ResNet-101 ImageNet

This Diploma Thesis addresses domain adaptation challenges in semantic segmentation by leveraging self-supervised depth estimation and auxiliary tasks. It proposes innovative strategies, including explicit task feature correlation learning and adaptation difficulty refinement based on depth predictions. These techniques enhance model performance, leading to a significant improvement of 4.79% mIoU.

In the last few years, Neural Networks have achieved overwhelming performance on many computer vision tasks. However, they require a large amount of annotated data to be trained properly. The image labeling process for semantic segmentation is particularly demanding as every pixel has to be annotated with one of the possible classes. For this reason, this thesis utilizes a source domain dataset where the semantics are similar and the data is automatically labeled. Domain adaptation for semantic segmentation aims to improve the model performance in the presence of a distribution shift between source and target domain. Leveraging the supervision from auxiliary tasks such as depth estimation has the potential to heal this shift. However, depth information is not always available. Therefore, guidance from self-supervised depth estimation is used, which can be applied to both domains to bridge the domain gap. On the one hand, it is proposed to explicitly learn the task feature correlation to strengthen the target semantic predictions with the help of target depth estimation. On the other hand, the depth prediction discrepancy from source and target depth decoders are used to approximate the pixel-wise adaptation difficulty. The adaptation difficulty, inferred from depth, is then used to refine the target semantic segmentation pseudo-labels. This thesis exploits three simple but crucial training strategies, in addition to the baseline, to stabilize the training and avoid overfitting to the source domain:

    (1) Rare Class Sampling on the source domain improves the quality of the pseudo-labels by mitigating the confirmation bias of self-training toward common classes
    (2) Learning Rate Warmup promote feature transfer from ImageNet pretraining
    (3) Rare Class Mixing helps the network using more pixels of rare classes next to target domain pixels

This approach improves baseline by 4.79% mIoU for GTA→Cityscapes and enables better learning, especially in the classes where there is increased segmentation difficulty.
