Top Skills in Data Engineer Job Listings

Python Airflow Selenium Docker PostgreSQL RegEx

This project consists of a robust Apache Airflow workflow automating the extraction of weekly job listing details from a popular job aggregator website through Selenium scraping. It uses PostgreSQL to store the scraped information in a useful schema containing historical data. Complemented by a Python script utilizing RegEx patterns, it can visually present the evolving demand for key skills in data engineering. Docker Compose seamlessly integrates these components, offering a concise yet powerful demonstration of automated data extraction, analysis, and visualization.

Have you ever wondered which high-value skills could boost your chances of landing a dream job? What if there was a seamless way to discover the key skills tailored to your job title? This project would offer you an innovative solution that unlocks insights and propels your career aspirations.

Leveraging Docker Compose to set up services for Apache Airflow, Selenium, and PostgreSQL enables seamless integration across various operating systems and facilitates straightforward deployment on diverse devices. Apache Airflow efficiently orchestrates a Directed Acyclic Graph (DAG) on a weekly schedule, utilizing Selenium to open an automated driver capable of interacting with and extracting information from any web page. Selenium workflow is carefully designed for the specific website structure, enabling smooth navigation through pages, capturing job links, and extracting details such as titles, links, and content. This extracted data is then stored in a sophisticated PostgreSQL schema including timestamps for exploration of historical trends.

Enhancing this automated web-scraping workflow is a Python script that transforms raw data into useful insights. It establishes a connection with the PostgreSQL database and leverages RegEx patterns to extract core skills from job descriptions. Matplotlib then creates plots that visualize the new aggregated data to showcase the top skills in data engineering job listings. Furthermore, it can conduct a comparative analysis between this week and the previous one, offering insights into the evolution of these skills over time.
